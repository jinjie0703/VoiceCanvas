# VoiceCanvas AI - 语音智能手绘白板

这是一个全语音控制的 AI 智能矢量白板与流程图绘制工具。通过前后端分离架构实现，利用大语言模型（LLM）与智能体（Agent）将模糊的语音指令解析为结构化的 JSON Action 序列，驱动前端的 TLDraw 手绘风格无限画布。

## 🌟 核心亮点

1. **TLDraw 手绘风画布**：采用现代、极简、优雅的马克笔手绘风格，开箱即用，支持各种矢量图形与平滑曲线连接。
2. **端侧 Web Speech API**：前端基于浏览器原生 Web Speech API 实现语音识别，不需要昂贵的云端语音听写，响应速度极快。
3. **语义网格对齐与空间感知**：避免大模型进行复杂的像素计算，由前端提供包含几何数据（x, y, w, h）的精准空间状态（State Observation），智能体可实现无碰撞、智能排版。
4. **多轮对话与自主纠错智能体 (Agent)**：后端集成 Eino 框架，构建 ReAct 智能体循环。支持多轮对话持续修改、复杂指令拆解与自动纠错。
5. **智能体操作日志 (Debug Panel)**：右侧面板实时滚动打印大模型的原生 JSON Action 和思考过程，硬核感和科技感直接拉满。
6. **Siri 风格动态音频反馈**：前端采用独立 CSS Modules 封装的高级光晕与渐变波形动画，提供“聆听中”与“思考中”等极其优雅的拟物化交互。
7. **无状态阅后即焚与会话持久化**：后端轻量级 WebSocket 通信，前端引入本地持久化存储（Local Storage）保障刷新不丢失工作流。
8. **高可用通信架构 (High-Availability WS)**：前端实现了带有指数退避的断线重连机制，结合基于队列的顺序动作执行（Queue-based Pipeline）与严格的 `__done__` 信号协议，完美解决状态冲突。
9. **流式 JSON 解析引擎 (Streaming Parser)**：后端重构了专为 LLM 输出设计的括号计数流式解析器（Brace-counting Scanner），杜绝残缺 JSON 导致的崩溃。
10. **多模态与视觉感知 (Multi-Modal Vision)**：打通视觉与空间的双向认知（集成多模态视觉大模型等能力），让智能体不仅“听得懂”，更“看得见”。
11. **极致工程化与解耦架构**：前端贯彻极致类型安全（无 `any` TS 标准），后端通过 `Engine` 编排与 `ClientSession` 接口彻底解耦业务逻辑与传输层，具备企业级扩展性。

## 🛠️ 支持的语音指令示例

- **创建图形与关系**：
  - *"在左上角画一个红色的便签"*
  - *"在右边画一个蓝色的圆形，写上'数据库'"*
  - *"把这两个图形用一条曲线连接起来"*
- **智能生成图片**：
  - *"帮我生成一张太空漫游的图片放在左下角"*
  - *"帮我搜索一张苹果的图片放在中间"*
- **修改与编辑图形**：
  - *"把那个红色的便签改成绿色"*
  - *"把那个圆里面的字改成后台服务器"*
  - *"把刚才画的矩形向右移动一点"*
- **删除与清空**：
  - *"删掉那个蓝色的圆形"*
  - *"清空画布"*
