# VoiceCanvas AI - 语音智能手绘白板

这是一个全语音控制的 AI 智能矢量白板与流程图绘制工具。通过前后端分离架构实现，利用大模型将模糊的语音指令解析为结构化的 JSON Action 序列，驱动前端的 TLDraw 手绘风格无限画布。

## 🌟 核心亮点
1. **TLDraw 手绘风画布**：采用现代、极简、优雅的马克笔手绘风格，开箱即用。
2. **端侧 Web Speech ASR**：前端基于浏览器原生 API，不需要昂贵的云端语音听写，响应速度极快。
3. **语义网格对齐 (九宫格)**：避免大模型进行复杂的像素计算，由前端将语义位置（如 `top_right`）自动映射为物理像素。
4. **意图拒识机制**：支持过滤背景聊天或闲聊，只在听到明确指令时才画图，防止演示翻车。
5. **智能体操作日志 (Debug Panel)**：右侧面板实时滚动打印大模型的原生 JSON Action，硬核感和科技感直接拉满。
6. **无状态阅后即焚**：后端无数据库开销，每次绘制请求都实时同步前端 Canvas Snapshot，降低网络吞吐。

## 🚀 快速启动指南

### 1. 启动 Go 后端
```bash
cd backend

# 配置您的 DashScope / ModelScope API 密钥
# (如果未配置，后端会自动进入 Mock 仿真模式，亦可完成基础演示)
export DASHSCOPE_API_KEY="您的_API_KEY"

# 运行服务 (监听 http://localhost:8080/ws)
go run main.go
```

### 2. 启动 React 前端
```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器 (监听 http://localhost:5173)
npm run dev
```

## 🛠️ 支持的语音指令示例

* **创建图形**：
  * *"在左上角画一个红色的便签"*
  * *"在右边画一个蓝色的圆形"*
  * *"在中间画个菱形，写上开始"*
* **修改图形**：
  * *"把那个红色的便签改成绿色"*
  * *"把那个圆里面的字改成后台服务器"*
* **删除与清空**：
  * *"删掉那个蓝色的圆形"*
  * *"清空画布"*
